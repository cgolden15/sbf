# Shady Brook Farm Bulk Material Calculator

A simple, user-friendly calculator to estimate bulk material quantities (mulch and stone) needed for landscaping projects.

## Features

✨ **Material Selection**
- Black Dyed Mulch ($44.99/yd³)
- Natural Triple Shred Mulch ($34.99/yd³)
- Premium Mulch - Natural Dark Brown ($54.99/yd³)
- 3/4" Clean Stone (~$49.99/ton)

📏 **Easy Measurements**
- Input length and width in feet
- Adjust depth in inches (defaults to recommended depths)
- Recommended defaults: 2-3" for mulch, 1.5-2" for stone

📊 **Instant Estimates**
- Calculates cubic yards or tonnage needed
- Shows detailed calculation breakdown
- Clearly marked as ESTIMATE only

🎨 **Farm-Themed Design**
- Rustic, earthy color palette matching Shady Brook Farm's brand
- Responsive design for mobile and desktop
- Clean, intuitive interface

## How It Works

1. **Select a material** from the dropdown
2. **Enter dimensions**: length × width (in feet) and depth (in inches)
3. **Click "Calculate Estimate"** to see how much material you need
4. **Review the results** - shows cubic yards/tons and a detailed breakdown
5. **Order directly** - click the button to order from Shady Brook Farm

## Calculation Details

### For Mulch (cubic yards)
```
Area (sq ft) × Depth (inches) ÷ 12 ÷ 27 = Cubic Yards
```

### For Stone (tons)
```
Cubic Yards × 1.5 tons per cubic yard = Tons
```
